# NCBI Sequence Fetcher & Analyzer

A Python notebook that fetches real gene sequences from NCBI
and analyzes their base composition.

## What it does
- Fetches DNA sequences from NCBI using accession numbers
- Computes GC%, AT% and base counts for each gene
- Saves results to CSV
- Plots GC content bar chart

## Libraries used
- Biopython — NCBI Entrez API
- pandas — data storage and CSV export
- matplotlib — visualization

## Genes analyzed
- BRCA1 — breast cancer susceptibility gene
- TP53  — tumor suppressor gene
- EGFR  — epidermal growth factor receptor

## How to run
Open in Google Colab and run all cells.

## Key concepts learned
- Biopython Entrez API
- SeqIO for sequence parsing
- gc_fraction for GC content
- pandas DataFrame and CSV export
- matplotlib bar charts
